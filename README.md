# BankingApp
 A simple Banking application built using Java Enterprise Edition integrated with MySQL database and Glassfish server. This application was developed using the NetBeans IDE 8.2 RC.

## Features

- Login
- Signup
- Deposit
- Withdraw
- Enquiry

## Technologies used

- NetBeans IDE 8.2 RC
- JavaEE
- MySQL
- GlassFish server
